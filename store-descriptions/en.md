Claude Export is a privacy-first browser extension for turning Claude conversations into organized knowledge and polished files. Sync Claude conversations directly to Notion or Obsidian, or export them locally in the format you need.

Key features:

1. One-click Notion / Obsidian sync: Send the current Claude conversation to a connected Notion database or Obsidian vault without copying and pasting.
2. Batch sync and batch export: Select multiple Claude conversations and sync them to Notion or Obsidian, or export them together as files. Process up to 10 chats in one batch.
3. Multiple export formats: Supports PDF, DOCX, Markdown, HTML, image, text and JSON formats.
4. Private local file generation: File conversion runs in your browser. Chat message bodies are not uploaded to a remote conversion server.
5. Professional themes and fields: Choose from document themes including Minimalist, Natural, Editorial, Terminal, Aurora and Oxford. Control titles, timestamps, platform labels and role fields.
6. Export only what matters: Select individual messages, question-and-answer turns or AI replies before syncing or exporting.
7. Report and developer tools: Build AI-only reports, preserve code blocks, and package code snippets into language-specific files.

How to use:

- Install the extension and open a conversation on Claude (claude.ai).
- Open Claude Export and choose Notion, Obsidian or a file format.
- For multiple conversations, open Batch Export, select the chats and choose a destination.
- Review the range and theme, then sync or download.

Claude Export is useful for research notes, project handoffs, knowledge bases, code archives, client deliverables, study materials and personal backups.
